## 装饰器的主要作用：
- 装饰类
- 装饰方法或属性
## 装饰类
```
@annotation
class Myclass {};
function annotation(target) {
  target.annotated = true;
}
/////////////////////////////////////////////////////////
@annotation
class Myclass {};
//相当于： 
class A {};
A = annotation(A) || A;
```
## 装饰方法或属性
```
class MyClass {
  @readonly
  method() {};
}
function readonly(target,name, descriotor) {
  descriptor.writable = false;
  return descriptor;
}
```
